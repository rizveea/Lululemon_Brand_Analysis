## Business Context
Understanding brand perception and engagement is crucial for marketing spend optimization and supplier brand evaluation. ## Project Highlights

### Problem Addressed
Built a **social media intelligence tool** to analyze Lululemon's Instagram presence by:
- Automating collection of posts, captions, and engagement metrics
- Correlating content features with user engagement
- Identifying high-performing content patterns

### Key Outcomes
- **Engagement Analytics**: Systematic tracking of likes, comments, and shares
- **Content Analysis**: Correlation between post attributes and engagement rates
- **Trend Identification**: Patterns in posting frequency, timing, and content types
- **Competitive Framework**: Methodology applicable to competitor brand analysis

## Methodology
1. **Data Collection**: Selenium-based scraping of Instagram posts and metrics
2. **Feature Extraction**: Caption text, hashtags, posting time, media type
3. **Engagement Analysis**: Statistical correlation with engagement metrics
4. **Visualization**: Trend charts and engagement dashboards
